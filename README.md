# Домашняя работа к лабораторной работе 2.
## Условия задачи:
Четыре человека пообедали в ресторане. Официант подал им счет на X руб каждому. Они решают  оставить официанту чаевые в размере 15% от счета. Составьте программу, которая выведет на экран сумму чаевых, которую получит официант.
## 1. Алгоритм и блок схема:
### Алгоритм:
1. **Начало**
2. Задать исходные данные:
   - `kl` = 4 - количество человек
   -  `x`= 1000 - счет на каждого человека
   -   `ch` = 0.15 - 15% чаевых
   -    `och` = общая сумма чаевых
3. Вычислить общую сумму чаевых:
   - `och` = `kl` * `x` * `ch`
4. Вывести результаты расчётов с подстановкой значений в текст.
5. **Конец**

## 2. Реализация программы:


      #define _CRT_SECURE_NO_WARNINGS
      #include <stdio.h>
      #include <locale.h>
      
      
      int main() {
          
          setlocale(LC_CTYPE, "");
      
          int kl = 4;  // количество человек
          int x;  // счет на каждого человека в рублях
          double ch = 0.15;  // 15% чаевых
          printf("Введите счет за обед на каждого человека: ");
          scanf("%d", &x);
          // общ сумма чаевых
          float och = kl * x * ch;
      
       
          printf("Счет на каждого человека в рублях: %.d руб\n", x);
          printf("Официант получит чаевых: %.2f руб.\n", och);
      
          return 0;
      }
